CSS; Cascading Style Sheets

What is CSS ?

CSS(Cascading Style Sheets) is a style sheet language used to describe the presentation of a document written in HTML or XML.

Who developed CSS ?

Primary Developer: Håkon Wium Lie

When CSS was developed ?

1989–1994: Håkon Wium Lie proposed the idea of separating content from presentation while working at CERN. 1994: The first official proposal for CSS was published by Håkon Wium Lie. Collaboration: Bert Bos, working on the Argo browser, collaborated with Håkon to expand and standardize the language.

Short Desc: Standardization: CSS was standardized by the World Wide Web Consortium (W3C) in 1996. Håkon Wium Lie became the first W3C CSS WorkingGroup Chair.

Intresting Facts of CSS

  • Håkon Wium Lie was inspired to create CSS while seeing how HTML lacked styling capabilities in the early web.
  • CSS allows cascading rules, meaning multiple styles can apply to the same element, with a defined order of precedence.
  • CSS is open standard, maintained and developed by W3C, allowing cross-browser compatibility.
