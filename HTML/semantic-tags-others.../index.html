Why we have to use Semantic tags ?

We have to use Semantic tags becasuse it helps Search engines to search easily and optimize own page and also helpful for blind people they use screen reader.

    Examples of Semantic tags-

  1. section tag
  2. main tag
  3. header tag
  4. footer tag
  5. title tag for define title of a page
  6. form is also Semantic tag

    Examples of Semantic tags(inline)

  1. em tags to emphasized text in a document
  2. strong tag show like its a important text in the document which helps screen-readers and search engines.
