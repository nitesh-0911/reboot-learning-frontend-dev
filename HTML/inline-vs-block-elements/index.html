What is inline and block elements ?

Examples of block elements-

  1. all headings
  2. div tag
  3. paragraph tag
  4. ...

Examples of inline elements-

  1. bold tag
  2. italic tag
  3. span tag(most commonly used)
  4. image tag
  5. ...

Block element takes full width of its parent, but inline element takes only content size width.
